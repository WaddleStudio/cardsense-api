# CardSense API

Decision-as-a-Service API for credit card reward optimization.

This API recommends the optimal credit card based on
transaction context using deterministic, explainable logic.

## What this repo does
- Exposes REST APIs for card recommendations
- Applies deterministic reward calculation rules
- Provides auditability and explainable results

## What this repo does NOT do
- No crawling or data extraction
- No LLM inference in request path
- No promotion full-text API
